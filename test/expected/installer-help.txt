Options:
  -d, --dest            Relative Destination path for the generated installer task file  [string] [required]
  -h, --hooks           Relative Path to directory holding git hook files generated by GenHook  [string] [required]
  -n, --name            Name to give the generated file  [string] [default: "installGitHooks.js"]
  -r, --taskRunnerRoot  Relative Path to your task runner sourcefile - i.e. path to gulpfile.js - defaults to CWD from where genhook command is ran.  [default: "./"]
  -t, --taskName        What you want the task to be named  [string] [default: "install-git-hooks"]
  --help                Show help  [boolean]

Examples:
  genhook installer -d ../gulp-tasks/ -h ../../git-hooks/ -n git-hook-installer.js -t installGitHooks

* Generated installer gulp task will require gulp-chmod and gulp-rename modules
