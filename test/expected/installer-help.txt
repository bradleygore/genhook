Options:
  -d, --dest      Relative Destination path for the generated installer task file  [required]
  -h, --hooks     Relative Path to directory holding git hook files generated by GenHook  [required]
  -n, --name      Name to give the generated file  [default: "installGitHooks.js"]
  -t, --taskName  What you want the task to be named  [default: "install-git-hooks"]
  --help          Show help  [boolean]

Examples:
  genhook installer -d ../gulp-tasks/ -h ../../git-hooks/ -n git-hook-installer.js -t installGitHooks
